# react-dynamic-title

> Example how to use a dynamic tab title on a react app

[Reference dev.to post](https://dev.to/luispa/how-to-add-a-dynamic-title-on-your-react-app----3l0j)

## 1. Install dependencies
Using npm
```bash
$ npm i 
```

OR Using yarn
```bash
$ yarn install
```

## 2. Run the example

Using npm
```bash
$ npm run start
```

OR Using yarn
```bash
$ yarn start
```
